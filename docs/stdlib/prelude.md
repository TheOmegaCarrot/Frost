# Prelude

The prelude is a set of functions written in Frost itself, loaded at startup.

## `inv`
`inv(f)`

## `reject`
`reject(structure, pred)`

## `curry`
`curry(f, ...outer)`

## `bcurry`
`bcurry(f, ...outer)`

## `collect`
`collect(...e)`

## `call`
`call(f, ...args)`

## `rev_args`
`rev_args(f)`

## `tap`
`tap(a, f)`

## `const`
`const(v)`

## `index`
`index(k)`

## `map_keys`
`map_keys(m, f)`

## `map_values`
`map_values(m, f)`

## `compose`
`compose(f, g, ...rest)`
