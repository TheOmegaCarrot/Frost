# Prelude

The prelude is a set of functions written in Frost itself, loaded at startup.
These functions are documented alongside the builtins they complement:

- [Collections](./collections.md): `reject`, `index`, `map_keys`, `map_values`
- [Functions](./functions.md): `inv`, `curry`, `bcurry`, `collect`, `call`, `rev_args`, `tap`, `const`, `compose`
