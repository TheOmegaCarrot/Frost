# Frost Challenge Scaffolds

These scripts are runnable assertion harnesses for language practice.

- Run one challenge:
  - `./build/frost frost-scripts/test-challenges/01-order-analytics.frst`
- Most scaffolds are expected to fail until TODOs are implemented.
